Object-Oriented Programming

a programming paradigm that uses abstraction to create models based on the real world

Class Defines the data type or characteristics of the Object

class Vehicle {

  // members or fields
  wheels: number;
  speed: number;
  color: string;

  // constructor
  constructor ( distance: number, duration: number, public destination: string ) {
    this.speed = distance / duration;
    this.wheels = 4
  };

  // methods
  honk() {
    return "At " + this.destination + ", the average speed was " + this.speed + " km/h";
  }
  hello() {
    return "Hello! I'm a " + this.wheels + "-wheeler vehicle!";
  }
}

Object An Instance of a Class

var car = new Vehicle(300, 5, "Brussels");

Property characteristics, fields or members of an object

wheels: number;
speed: number;
color: string;

Method function or capability of an object

honk() {
  return "At " + this.destination + ", the average speed was " + this.speed + " km/h";
}
hello() {
  return "Hello! I'm a " + this.color + "vehicle!";
}

Constructor a function called at instantiation

constructor ( distance: number, duration: number, public destination: string ) {
  this.speed = distance / duration;
  this.color = "white"
};

Inheritance a derived class can inherit characteristics from a base class

class Bicycle extends Vehicle {

  constructor ( distance: number, duration: number, public destination: string ) {
    super( distance, duration, destination ); // calls the base class constructor
    this.speed = distance / duration;
    this.wheels = 2
  };

  hello() {
    return "Hello! I'm a " + this.wheels + "-wheeler vehicle!";
  }
}
